# Aether Language Extension

Local VS Code language support for `.ath` files.

Includes:

- Syntax highlighting
- Language configuration
- Snippets
- Command Palette action: `Aether: New Project`
- Command Palette action: `Aether: Configure Run/Debug For Workspace`
- Command Palette action: `Aether: Run Current File`
- Command Palette action: `Aether: Debug Current File`
